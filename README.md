# gei-776
Projet GEI 776

## Requirements

* Java 1.8
* Latest version of Docker

## Running the app

First, start the database:

```
> docker-compose up
```

Then, start the app:

```
> mvnw spring-boot:run
```

## Using the app

Go to [http://localhost:8080]() and connect using the following credentials:

```
User:      admin@syntaxerror.blog
Password:  password
```
